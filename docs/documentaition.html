معرفی قالب هاستینگ

این قالب به‌صورت کامل و حرفه‌ای برای سایت‌های هاستینگ طراحی شده است تا بتوانید با ظاهری زیبا و امکانات کافی، سایت خود را راه‌اندازی کنید.

نصب و راه‌اندازی

برای نصب قالب، کافیست فایل‌های قالب را در پوشه اصلی وب‌سرور خود آپلود کنید و سپس از طریق مرورگر به آدرس سایت مراجعه نمایید.

ftp upload / your hosting root folder

اطمینان حاصل کنید که پوشه assets و فایل‌های CSS و JS به درستی بارگذاری شده باشند.

نحوه استفاده

قالب شامل صفحات مختلفی مانند صفحه اصلی، داشبورد کاربری، ورود و ثبت‌نام، جزئیات سرویس و ... می‌باشد که هر کدام عملکرد مشخصی دارند.

  • صفحه اصلی: معرفی خدمات و جلب توجه کاربر.
  • داشبورد کاربری: مدیریت سرویس‌ها و مشاهده وضعیت حساب.
  • ورود / ثبت‌نام: مدیریت حساب کاربری.
  • صفحه ۴۰۴: نمایش خطای صفحه پیدا نشد با انیمیشن حرفه‌ای.

شخصی‌سازی قالب

برای تغییر رنگ‌ها، فونت‌ها و تصاویر قالب می‌توانید فایل assets/css/style.css را ویرایش کنید یا بخش‌های HTML را مطابق نیاز خود تغییر دهید.

همچنین می‌توانید انیمیشن‌ها و اسکریپت‌های جاوااسکریپت را به دلخواه اضافه یا تغییر دهید.
